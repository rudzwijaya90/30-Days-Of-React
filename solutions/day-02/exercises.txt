Exercises: JSX
What is an HTML element?
1. An HTML element has an opening and closing tags, content, and attribute in the opening tag. However, some HTML elements may not have content and a closing tag - they are self closing elements.

How to write a self closing HTML element?
2. Instead of using a separate closing tag (like </div>), self-closing elements are "closed" within the opening tag itself by adding a slash (/) before the closing angle bracket (>).

What is an HTML attribute? Write some of them
3. An HTML attribute is a modifier or additional information added to an HTML element to provide more details about how the element should behave, appear, or interact with the user.
common HTML attribute: id, class, type, text, href, src, alt, style, title, name, placeholder, type, value, target, rel, rowspan, colspan

What is JSX?
4. JSX stands for JavaScript XML. JSX allows us to write HTML elements with JavaScript code. 

What is babel?
5. Babel is a library which transpiles JSX to pure JavaScript and latest JavaScript to older version

What is a transpiler?
6. A transpiler (short for source-to-source compiler) is a type of compiler that takes source code written in one programming language and translates it into source code in another programming language, often with the goal of enabling compatibility or adding features not directly supported by the target language.

Exercises: JSX Elements
What is a JSX element?
1. JSX element could be a single HTML element or many HTML elements wrapped in a parent HTML element.
A JSX element (short for JavaScript XML) is a syntax extension for JavaScript, commonly used with React, which allows you to write HTML-like code directly within JavaScript. JSX elements look very similar to HTML tags but are actually JavaScript objects behind the scenes. JSX allows developers to describe the structure of the user interface in a declarative way, combining the best features of HTML and JavaScript into a single, readable syntax.
